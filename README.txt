En este proyecto se realizaron dos ejercicios:
En el primero se calculó un sueldo y a partir de condicionales se buscó la categoría respectiva. En este ejercicio se aplicaron las nociones básicas de Python.
Para el segundo ejercicio se utilizó una base de datos elegida sin ningún requisito, se trabajó con la librería Pandas optimizando su visualización.
